# gojs-seatingchart

### By Northwoods Software for [GoJS](https://gojs.net)

This project provides a basic example of using GoJS in a Vue app.

## Installation

Start by running `npm install` to install all necessary dependencies.
`npm run dev` runs the app in the development mode.

## GoJS in Vue

This sample demonstrates how to use GoJS diagrams within a Vue application. It contains:

- A guest diagram to drag and drop guests to the main diagram
- A double click handler to add tables
- An editor box to make updates to the current selection, opened by pressing `E` or double clicking a table or guest
- The ability to import and export model JSON
